### baseline_representations folder:
Contains the code for two baseline methods, the STFT and the CFT and the corresponding inverse transform.

### clusterability:
Contains the code for computing the normalized cut loss for mixtures represented in the STFT, CQT, CFT, and MCFT domains.

### separability:
Contains the code for computing BSS-Eval measures for sources that are separated through ideal binary masking in the STFT, CQT, CFT, and MCFT domains.
